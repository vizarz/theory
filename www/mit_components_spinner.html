Spinner

Компонент Spinner представляет собой выпадающий список, который позволяет пользователю выбрать один элемент из предопределенного набора значений. Spinner занимает мало места на экране, так как отображает только текущий выбранный элемент, а полный список значений появляется только при взаимодействии с компонентом.

Spinner часто используется для предоставления пользователю выбора из фиксированного набора опций, таких как категории, настройки или параметры фильтрации.

    Методы

    DisplayDropdown()
    Отображает выпадающий список с элементами Spinner для выбора пользователем.

    Свойства

    BackgroundColor
    Определяет цвет фона компонента Spinner.
    Elements
    Определяет список элементов для отображения в выпадающем списке.
    ElementsFromString
    Определяет элементы списка в виде строки, разделенной запятыми. Например, "Элемент1, Элемент2, Элемент3".
    Enabled
    Определяет, включен ли компонент. Если установлено значение False, пользователь не сможет взаимодействовать с компонентом.
    FontBold
    Определяет, отображается ли текст полужирным шрифтом.
    FontItalic
    Определяет, отображается ли текст курсивом.
    FontSize
    Определяет размер шрифта текста, отображаемого в компоненте.
    FontTypeface
    Определяет гарнитуру шрифта для текста.
    Height
    Определяет высоту компонента в пикселях.
    Prompt
    Определяет текст подсказки, который отображается в выпадающем списке. Этот текст виден, когда пользователь открывает список, но не является выбираемым элементом.
    Selection
    Определяет текущий выбранный элемент в Spinner. Это свойство может быть установлено для программного выбора элемента.
    SelectionIndex
    Определяет индекс текущего выбранного элемента в списке (начиная с 1). Значение 0 означает, что выбор не сделан.
    TextColor
    Определяет цвет текста, отображаемого в компоненте.
    Visible
    Определяет, виден ли компонент на экране.
    Width
    Определяет ширину компонента в пикселях.

    События

    AfterSelecting(selection)
    Событие, которое происходит после того, как пользователь выбирает элемент из выпадающего списка. Параметр selection содержит текст выбранного элемента.
