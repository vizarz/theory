ListView

Компонент ListView – это видимый элемент, который отображает список текстовых и графических элементов на экране. Простые списки строк можно задать с помощью свойства ElementsFromString. Более сложные списки, содержащие несколько строк и/или изображений, можно создать, используя свойства ListData и ListViewLayout.

Внимание: Этот компонент будет работать некорректно на экранах с возможностью прокрутки, если его высота установлена в режим Fill Parent.

    Методы

    AddItem(mainText, detailText, imageName)
    Добавляет новый элемент в список.
    AddItemAtIndex(index, mainText, detailText, imageName)
    Добавляет новый элемент в список по заданному индексу.
    AddItems(itemsList)
    Добавляет новые элементы в список.
    AddItemsAtIndex(index, itemsList)
    Добавляет новые элементы в список по заданному индексу.
    CreateElement(mainText, detailText, imageName)
    Создает элемент списка с указанным основным текстом, дополнительным текстом и именем изображения.
    GetDetailText(listElement)
    Возвращает дополнительный текст (Detail Text) для данного элемента ListView.
    GetImageName(listElement)
    Возвращает имя файла изображения, загруженного в Media, для данного элемента ListView.
    GetMainText(listElement)
    Возвращает основной текст (Main Text) для данного элемента ListView.
    RemoveItemAtIndex(index)
    Удаляет элемент из списка по заданному индексу.

    Свойства

    BackgroundColor
    Задает цвет фона ListView.
    BounceEdgeEffect
    Устанавливает функцию эффекта отскока. Значение true включает эффект, false – отключает.
    DividerColor
    Задает цвет разделителя ListView.
    DividerThickness
    Определяет толщину разделителя в ListView.
    ElementColor
    Задает цвет элемента ListView.
    ElementCornerRadius
    Определяет радиус скругления углов элемента списка.
    ElementMarginsWidth
    Задает ширину отступов элемента ListView.
    Elements
    Задает список вариантов для отображения в ListView.
    ElementsFromString
    Задает список вариантов как строку с элементами, разделенными запятыми, например: Cheese,Fruit,Bacon,Radish.
    FontSizeDetail
    Определяет размер шрифта дополнительного текста элементов ListView.
    FontTypeface
    Задает тип шрифта основного текста ListView (default, serif, sans serif или monospace).
    FontTypefaceDetail
    Задает тип шрифта дополнительного текста ListView (default, serif, sans serif или monospace).
    Height
    Задает вертикальную высоту ListView в пикселях.
    HeightPercent
    Задает вертикальную высоту ListView как процент от высоты экрана.
    HintText
    Устанавливает подсказку в строке фильтра.
    ImageHeight
    Задает высоту изображения в макетах ListView, содержащих изображения.
    ImageWidth
    Задает ширину изображения в макетах ListView, содержащих изображения.
    ListData
    Задает данные для отображения в элементах ListView. Это свойство определяет элементы, указанные в ListViewLayout. Например, если выбран макет "Image,MainText", оно позволяет задать элементы, каждый из которых содержит имя файла для Image и строку для MainText. Свойство доступно только в Designer.
    ListViewLayout
    Задает тип макета для строки ListView.
    Orientation
    Определяет ориентацию макета. Допустимые значения: Vertical – элементы располагаются вертикально друг за другом; Horizontal – отображается один элемент за раз с возможностью пролистывания влево или вправо.
    Selection
    Возвращает текст выбранного элемента ListView по позиции, указанной в SelectionIndex.
    SelectionColor
    Задает цвет выбранного элемента ListView.
    SelectionDetailText
    Возвращает дополнительный (Secondary/Detail) текст элемента ListView по позиции, указанной в SelectionIndex.
    SelectionIndex
    Устанавливает индекс выбранного элемента (начиная с 1). Если элемент не выбран, значение равно 0. Если заданное значение меньше 1 или больше числа элементов, SelectionIndex устанавливается в 0, а Selection — в пустую строку.
    ShowFilterBar
    Устанавливает видимость строки фильтра. Значение true – строка видна, false – скрыта.
    TextColor
    Задает цвет основного текста элементов ListView.
    TextColorDetail
    Задает цвет дополнительного текста в макете ListView.
    TextSize
    Определяет размер шрифта основного текста элемента ListView.
    Visible
    Определяет, должен ли ListView отображаться на экране (true – виден, false – скрыт).
    Width
    Задает горизонтальную ширину ListView в пикселях.
    WidthPercent
    Задает горизонтальную ширину ListView как процент от ширины экрана.

    События

    AfterPicking()
    Простое событие, которое срабатывает после выбора элемента из списка. Выбранный элемент доступен через свойство Selection.
