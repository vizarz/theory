Проект "Музыкальный плеер"

Цель: Создать простой музыкальный плеер, который воспроизводит твои любимые мелодии! Ты сможешь включать музыку, останавливать её и выбирать разные песни.

Этот проект познакомит тебя с работой с аудиофайлами в приложении, созданием пользовательского интерфейса для воспроизведения музыки и управлением компонентами воспроизведения звука.

Чему ты научишься:

  • Добавлять аудио файлы в приложение
  • Управлять воспроизведением звука
  • Создавать список воспроизведения
  • Настраивать громкость воспроизведения

Шаг 1: Добавление аудио ресурсов

  • Откройте MIT App Inventor и создайте новый проект с названием "MusicPlayer".
  • Перед началом работы с интерфейсом, нам нужно добавить аудиофайлы для воспроизведения. В панели "Palette" (Палитра) справа найдите раздел "Media" (Медиа).
  • Перетащите компонент Player (Проигрыватель) на экран вашего приложения. Обратите внимание, что этот компонент появится в разделе "Non-visible components" (Невидимые компоненты) внизу экрана.
  • Теперь нам нужно загрузить аудиофайлы в проект. Нажмите на "Upload File..." (Загрузить файл...) в разделе "Media" (Медиа).
  • Загрузите несколько MP3-файлов с музыкой. Для учебного проекта можно использовать короткие аудиофайлы или звуковые эффекты.
  • После загрузки вы увидите ваши аудиофайлы в списке медиа-файлов. Они будут автоматически доступны в вашем проекте.
Совет: Используйте файлы MP3 небольшого размера, чтобы ваше приложение не было слишком тяжелым. Для учебных целей достаточно файлов размером до 1 МБ.

Шаг 2: Создание интерфейса плеера

  • Теперь создадим интерфейс для нашего музыкального плеера. В разделе Layout (Макет) перетащите компонент VerticalArrangement (Вертикальное расположение) на экран.
  • Настройте свойства VerticalArrangement1:
    • Width (Ширина): Fill parent (Заполнить родителя)
    • Height (Высота): Fill parent (Заполнить родителя)
    • BackgroundColor (Цвет фона): Выберите приятный цвет для фона
  • Добавим заголовок плеера. Из раздела User Interface (Интерфейс пользователя) перетащите компонент Label (Метка) в VerticalArrangement1.
  • Настройте свойства Label1:
    • Text (Текст): "Мой музыкальный плеер"
    • TextColor (Цвет текста): Выберите контрастный цвет
    • FontSize (Размер шрифта): 20
    • FontBold (Жирный шрифт): True (Да)
  • Добавим метку для отображения названия текущей песни. Перетащите еще одну Label (Метку) в VerticalArrangement1 под заголовком.
  • Настройте свойства Label2:
    • Text (Текст): "Сейчас играет: ничего не выбрано"
    • Width (Ширина): Fill parent (Заполнить родителя)
    • TextAlignment (Выравнивание текста): Center (По центру)
    • FontSize (Размер шрифта): 16
  • Теперь добавим кнопки управления воспроизведением. Из раздела Layout (Макет) перетащите компонент HorizontalArrangement (Горизонтальное расположение) в VerticalArrangement1 под метками.
  • Настройте свойства HorizontalArrangement1:
    • Width (Ширина): Fill parent (Заполнить родителя)
    • AlignHorizontal (Горизонтальное выравнивание): Center (По центру)
  • Из раздела User Interface (Интерфейс пользователя) перетащите три компонента Button (Кнопка) в HorizontalArrangement1.
  • Настройте свойства первой кнопки (Button1):
    • Text (Текст): "◀" (символ "назад")
    • Width (Ширина): 60 пикселей
    • FontSize (Размер шрифта): 20
  • Настройте свойства второй кнопки (Button2):
    • Text (Текст): "▶" (символ "воспроизвести")
    • Width (Ширина): 60 пикселей
    • FontSize (Размер шрифта): 20
  • Настройте свойства третьей кнопки (Button3):
    • Text (Текст): "▶▶" (символ "вперед")
    • Width (Ширина): 60 пикселей
    • FontSize (Размер шрифта): 20
  • Добавим ползунок громкости. Из раздела User Interface (Интерфейс пользователя) перетащите компонент Slider (Ползунок) в VerticalArrangement1 под кнопками управления.
  • Настройте свойства Slider1:
    • Width (Ширина): Fill parent (Заполнить родителя)
    • MinValue (Минимальное значение): 0
    • MaxValue (Максимальное значение): 100
    • ThumbPosition (Положение ползунка): 50
Совет: Вместо текстовых символов для кнопок вы можете использовать изображения. Загрузите иконки воспроизведения, паузы и перемотки, а затем установите их как фон кнопок с помощью свойства Image.

Шаг 3: Добавление списка песен

  • Теперь добавим список песен, из которого пользователь сможет выбирать музыку для воспроизведения. Из раздела User Interface (Интерфейс пользователя) перетащите компонент ListView (Список) в VerticalArrangement1 под ползунком громкости.
  • Настройте свойства ListView1:
    • Width (Ширина): Fill parent (Заполнить родителя)
    • Height (Высота): 200 пикселей
    • BackgroundColor (Цвет фона): Выберите контрастный цвет к фону приложения
  • Теперь перейдем во вкладку Blocks (Блоки) для инициализации списка песен.
  • Первым делом создадим глобальную переменную для хранения списка песен. В категории Variables (Переменные) выберите блок initialize global name to (инициализировать глобальную переменную).
  • Измените имя переменной на songsList (список песен).
  • В категории Lists (Списки) найдите блок create empty list (создать пустой список) и соедините его с переменной songsList.
  • Теперь создадим еще одну глобальную переменную для хранения путей к аудиофайлам. В категории Variables выберите еще один блок initialize global name to.
  • Измените имя переменной на songPaths (пути к песням).
  • Соедините еще один блок create empty list с переменной songPaths.
  • Теперь заполним списки названиями песен и путями к файлам. В категории Lists найдите блок add items to list (добавить элементы в список).
  • Перетащите блок add items to list под блок инициализации переменной songsList.
  • В первое поле блока add items to list поместите переменную songsList. Затем добавьте названия песен, которые вы загрузили в проект, используя текстовые блоки из категории Text. Например:
    • "Песня 1"
    • "Песня 2"
    • "Песня 3"
  • Аналогично заполните список songPaths. Используйте блок add items to list и добавьте имена файлов для каждой песни, например:
    • "song1.mp3"
    • "song2.mp3"
    • "song3.mp3"
  • Теперь нам нужно заполнить ListView1 названиями песен. В категории ListView1 найдите блок ListView1.Elements (элементы списка).
  • Перетащите блок set ListView1.Elements to после заполнения списков песен.
  • В поле блока set ListView1.Elements to поместите переменную songsList.
Важно: Убедитесь, что порядок песен в списке songsList соответствует порядку файлов в списке songPaths. Это необходимо для правильного воспроизведения выбранной песни.

Шаг 4: Программирование кнопок управления

  • Теперь запрограммируем логику работы нашего музыкального плеера. Сначала создадим глобальную переменную для хранения текущего индекса песни.
  • В категории Variables (Переменные) выберите блок initialize global name to и назовите переменную currentSongIndex.
  • Установите начальное значение currentSongIndex равным 1 (первая песня в списке).
  • Теперь запрограммируем выбор песни из списка. В категории ListView1 найдите блок when ListView1.AfterPicking (после выбора элемента из списка).
  • Внутри этого блока нам нужно:
    1. Получить индекс выбранной песни
    2. Обновить currentSongIndex
    3. Установить источник для Player1
    4. Начать воспроизведение
  • Сначала получим индекс выбранной песни. В категории ListView1 найдите блок ListView1.Selection (выбранный элемент).
  • В категории Lists найдите блок index in list (индекс в списке). В первое поле поместите songsList, а во второе - ListView1.Selection.
  • Обновите currentSongIndex, используя блок set currentSongIndex to и результат блока index in list.
  • Теперь установим источник для Player1. В категории Player1 найдите блок set Player1.Source to.
  • Для получения пути к файлу используйте блок select list item из категории Lists. В первое поле поместите songPaths, а во второе - currentSongIndex.
  • Добавьте блок call Player1.Start для начала воспроизведения.
  • Обновите текст Label2, чтобы показать название текущей песни. Используйте блок set Label2.Text to и объедините текст "Сейчас играет: " с выбранным элементом из списка songsList.
  • Теперь запрограммируем кнопку воспроизведения/паузы (Button2). Найдите блок when Button2.Click.
  • Внутри этого блока нам нужно проверить, воспроизводится ли музыка сейчас, и соответственно либо приостановить, либо продолжить воспроизведение.
  • Используйте блок if then else из категории Control.
  • В условие поместите блок Player1.IsPlaying из категории Player1.
  • В блок then (тогда) добавьте:
    1. call Player1.Pause для приостановки воспроизведения
    2. set Button2.Text to "▶" (символ воспроизведения)
  • В блок else (иначе) добавьте:
    1. call Player1.Start для начала воспроизведения
    2. set Button2.Text to "❚❚" (символ паузы)
  • Запрограммируем кнопку "предыдущая песня" (Button1). Найдите блок when Button1.Click.
  • Внутри этого блока нам нужно:
    1. Уменьшить currentSongIndex на 1
    2. Проверить, не вышли ли мы за границы списка
    3. Установить новый источник для Player1
    4. Начать воспроизведение
  • Используйте блок set currentSongIndex to и уменьшите значение на 1 с помощью блока currentSongIndex - 1.
  • Добавьте проверку границ с помощью блока if then. В условие поместите блок currentSongIndex < 1.
  • В блок then добавьте set currentSongIndex to и установите значение равным длине списка songsList с помощью блока length of list.
  • Аналогично запрограммируйте кнопку "следующая песня" (Button3), но увеличивайте currentSongIndex на 1 и проверяйте, не превысили ли вы длину списка.
  • Наконец, запрограммируем ползунок громкости. Найдите блок when Slider1.PositionChanged.
  • Внутри этого блока добавьте set Player1.Volume to и установите значение равным Slider1.ThumbPosition / 100 (поскольку громкость в Player1 устанавливается в диапазоне от 0 до 1).
Совет: Для улучшения пользовательского опыта добавьте обработку события Player1.Completed, чтобы автоматически переходить к следующей песне, когда текущая закончится.

  • Проверьте основные функции:
    • Выбор песни из списка
    • Воспроизведение и пауза
    • Переключение между песнями с помощью кнопок "вперед" и "назад"

  • Возможные улучшения приложения:
    • Добавьте индикатор прогресса воспроизведения
    • Реализуйте функцию перемотки песни
    • Добавьте возможность создания плейлистов
    • Добавьте обложки альбомов для песен
    • Реализуйте режимы случайного воспроизведения и повтора
