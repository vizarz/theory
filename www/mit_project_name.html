Проект "Привет, мир!"

Цель: Создать простое мобильное приложение, которое при нажатии на кнопку выводит сообщение "Привет, мир!".
Этот проект поможет познакомиться с основными элементами MIT App Inventor: Добавление кнопки (Button) и надписи (Label) на экран. Программирование реакции на нажатие кнопки с помощью блоков.

Следуйте приведенным ниже шагам для реализации проекта. После каждого выполненного шага нажмите кнопку «Пометить как выполнено».

  • Откройте MIT App Inventor.
  • Войдите в свою учетную запись.
  • Выберите "Projects" в верхнем меню.
  • Нажмите "Create Apps" → "Start new project".
  • Введите название проекта, например, "HelloWorld", и нажмите OK.

    • Создайте новый проект в Unity и настройте его основные параметры.

    Добавьте основные компоненты, настройте сцены, реализуйте игровую логику и т.д.
