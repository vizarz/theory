PasswordTextBox

Компонент PasswordTextBox представляет собой поле для ввода текста, где символы маскируются для обеспечения конфиденциальности при вводе паролей. Этот компонент имеет те же функции, что и обычный TextBox, но скрывает введенный текст, отображая точки или звездочки вместо символов.

Пользователи могут вводить текст, взаимодействуя непосредственно с компонентом на экране устройства. Приложение может также задать содержимое компонента, используя свойство Text.

    Методы

    RequestFocus()
    Запрашивает фокус для этого компонента. Когда компонент получает фокус, он активирует клавиатуру для ввода текста.
    HideKeyboard()
    Скрывает клавиатуру. Только для компонентов Android версии 3.0 или выше.

    Свойства

    BackgroundColor
    Определяет цвет фона компонента PasswordTextBox.
    Enabled
    Определяет, включен ли компонент. Если значение установлено как False, компонент будет отключен и пользователь не сможет с ним взаимодействовать.
    FontBold
    Определяет, отображается ли текст полужирным шрифтом.
    FontItalic
    Определяет, отображается ли текст курсивом.
    FontSize
    Определяет размер шрифта текста, отображаемого в компоненте.
    FontTypeface
    Определяет гарнитуру шрифта для текста.
    Height
    Определяет высоту компонента.
    Hint
    Текст, который будет отображаться в компоненте, когда он пуст. Служит подсказкой пользователю о том, какую информацию следует ввести.
    NumbersOnly
    Определяет, ограничен ли ввод только цифрами. По умолчанию имеет значение False, что позволяет вводить любые символы. Если установлено значение True, компонент примет только цифры.
    PasswordVisible
    Определяет, видим ли текст пароля. По умолчанию имеет значение False, что означает, что введенные символы будут скрыты. Если установлено значение True, текст будет отображаться обычным образом.
    TextAlignment
    Определяет выравнивание текста в компоненте. Возможные значения: 0 (по левому краю), 1 (по центру), 2 (по правому краю).
    TextColor
    Определяет цвет текста, отображаемого в компоненте.
    Text
    Определяет текст, отображаемый в компоненте.
    Visible
    Определяет, видим ли компонент.
    Width
    Определяет ширину компонента.

    События

    GotFocus()
    Событие, которое происходит, когда компонент получает фокус.
    LostFocus()
    Событие, которое происходит, когда компонент теряет фокус.
    OnTextChanged()
    Событие, которое происходит, когда пользователь изменяет текст в компоненте.
