Slider

Компонент Slider представляет собой ползунок, который позволяет пользователю выбрать значение путем перемещения "бегунка" между минимальным и максимальным значениями. Компонент имеет настраиваемый диапазон значений и может использоваться для различных функций управления, таких как настройка громкости, яркости или любых других параметров, требующих выбора значения из диапазона.

Когда пользователь перемещает бегунок, происходит событие PositionChanged, которое может быть использовано для обновления других компонентов или выполнения действий в зависимости от выбранного значения.

    Свойства

    BackgroundColor
    Определяет цвет фона ползунка.
    ColorLeft
    Определяет цвет ползунка слева от бегунка.
    ColorRight
    Определяет цвет ползунка справа от бегунка.
    Enabled
    Определяет, включен ли ползунок. Если установлено значение False, пользователь не сможет взаимодействовать с ползунком.
    Height
    Определяет высоту компонента в пикселях.
    MaxValue
    Определяет максимальное значение ползунка. Начальное значение: 100.
    MinValue
    Определяет минимальное значение ползунка. Начальное значение: 0.
    ThumbEnabled
    Определяет, можно ли перемещать бегунок. Если установлено значение False, пользователь не сможет изменить позицию бегунка.
    ThumbPosition
    Определяет текущую позицию бегунка на ползунке. Значение должно быть между MinValue и MaxValue.
    Visible
    Определяет, виден ли компонент на экране.
    Width
    Определяет ширину компонента в пикселях.

    События

    PositionChanged(thumbPosition)
    Событие, которое происходит, когда пользователь изменяет положение бегунка. Параметр thumbPosition содержит новое значение позиции бегунка.
