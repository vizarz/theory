Проект "Магический шар"

Цель: Создать приложение с волшебным шаром, которое отвечает на вопросы пользователя, когда он встряхивает телефон.

Этот проект научит тебя работать с датчиком движения (акселерометром), создавать списки данных и выбирать случайные элементы из списка.

Чему ты научишься:

  • Распознавать встряхивание телефона
  • Выбирать случайные ответы из списка
  • Работать со списками данных

Шаг 1: Создание списка ответов

  • Откройте MIT App Inventor и создайте новый проект с названием "MagicBall".
  • Первым делом создадим список возможных ответов волшебного шара. Перейдите во вкладку Blocks в правом верхнем углу экрана.
  • Нажмите на категорию Variables (Переменные) в левой панели, затем выберите initialize global name to (инициализировать глобальную переменную).
  • Измените имя переменной на answers (ответы).
  • Теперь нам нужно создать список ответов. В левой панели выберите категорию Lists (Списки) и найдите блок create empty list (создать пустой список).
  • Перетащите блок create empty list и соедините его с переменной answers.
  • Теперь добавим ответы в список. В категории Lists найдите блок add items to list (добавить элементы в список).
  • Перетащите блок add items to list в рабочую область и соедините его с блоком инициализации переменной answers.
  • В первое поле блока add items to list перетащите вашу переменную answers из категории Variables.
  • Теперь добавим ответы. В категории Text выберите блок с текстом и добавьте его в блок add items to list. Напишите первый ответ, например, "Да!".
  • Нажмите на синий значок "+" в блоке add items to list, чтобы добавить еще поля для ответов.
  • Добавьте еще несколько ответов, например:
    • "Нет."
    • "Возможно..."
    • "Спроси позже"
    • "Однозначно!"
    • "Не рассчитывай на это"
    • "Мои источники говорят - да"
    • "Вероятность невелика"
Совет: Вы можете добавить любое количество ответов! Чем больше ответов, тем разнообразнее будет реагировать ваш магический шар.

Шаг 2: Добавление датчика движения

  • Теперь перейдите обратно во вкладку Designer для добавления компонентов интерфейса и датчиков.
  • В левой панели найдите раздел Sensors (Датчики).
  • Перетащите компонент AccelerometerSensor (Акселерометр) на экран приложения. Обратите внимание, что этот компонент появится в разделе "Non-visible components" (Невидимые компоненты) внизу экрана.
  • Выберите добавленный компонент AccelerometerSensor1 и настройте его свойства в правой панели:
    • Enabled (Включен): True (Да) - чтобы датчик работал
    • MinimumInterval (Минимальный интервал): 400 - интервал обновления в миллисекундах
    • Sensitivity (Чувствительность): Установите значение от 20 до 25 - чувствительность определения встряхивания
Внимание: Акселерометр работает только на реальных устройствах! При тестировании на эмуляторе вам понадобится использовать кнопку вместо встряхивания.

Шаг 3: Создание интерфейса волшебного шара

  • Теперь создадим интерфейс нашего приложения. В разделе Layout (Макет) перетащите компонент VerticalArrangement (Вертикальное расположение) на экран.
  • Настройте свойства VerticalArrangement1:
    • Width (Ширина): Fill parent (Заполнить родителя)
    • Height (Высота): Fill parent (Заполнить родителя)
    • AlignHorizontal (Горизонтальное выравнивание): Center (По центру)
    • AlignVertical (Вертикальное выравнивание): Center (По центру)
    • BackgroundColor (Цвет фона): Выберите тёмно-синий цвет
  • Добавим заголовок. Из раздела User Interface (Интерфейс пользователя) перетащите компонент Label (Метка) в VerticalArrangement1.
  • Настройте свойства Label1:
    • Text (Текст): "Магический шар"
    • TextColor (Цвет текста): White (Белый)
    • FontSize (Размер шрифта): 24
    • FontBold (Жирный шрифт): True (Да)
  • Теперь создадим сам шар. Из раздела Drawing and Animation (Рисование и анимация) перетащите компонент Canvas (Холст) в VerticalArrangement1 под Label1.
  • Настройте свойства Canvas1:
    • Width (Ширина): 300 пикселей
    • Height (Высота): 300 пикселей
    • BackgroundColor (Цвет фона): None (Нет) или тот же цвет, что и фон VerticalArrangement1
  • Теперь добавим изображение шара. Из раздела Drawing and Animation перетащите компонент Ball (Мяч) на Canvas1.
  • Настройте свойства Ball1:
    • Radius (Радиус): 100 - чтобы шар был большим
    • PaintColor (Цвет): Black (Чёрный)
    • X: 150 - центр холста по горизонтали
    • Y: 150 - центр холста по вертикали
  • Теперь добавим место для отображения ответа. Из раздела User Interface перетащите компонент Label (Метка) на Canvas1.
  • Настройте свойства Label2:
    • Text (Текст): "Встряхни телефон, чтобы получить ответ"
    • TextColor (Цвет текста): White (Белый)
    • FontSize (Размер шрифта): 18
    • X: 150 - центр холста по горизонтали
    • Y: 150 - центр холста по вертикали
  • Для тестирования на эмуляторе добавим кнопку. Из раздела User Interface перетащите компонент Button (Кнопка) в VerticalArrangement1 под Canvas1.
  • Настройте свойства Button1:
    • Text (Текст): "Встряхнуть (для эмулятора)"
    • Width (Ширина): Automatic (Автоматически)
    • FontSize (Размер шрифта): 16
Совет: Убедитесь, что Label2 (метка для ответа) находится над Ball1 (шаром). Для этого вы можете изменить порядок компонентов в панели Components, перетаскивая их вверх или вниз.

Шаг 4: Добавление случайного выбора

  • Теперь запрограммируем логику приложения. Вернитесь во вкладку Blocks.
  • Сначала запрограммируем реакцию на встряхивание. В левой панели найдите компонент AccelerometerSensor1 и выберите блок when AccelerometerSensor1.Shaking (когда акселерометр обнаруживает встряхивание).
  • Теперь нам нужно выбрать случайный ответ из списка. В категории Lists (Списки) найдите блок select list item (выбрать элемент списка).
  • Перетащите блок select list item в рабочую область.
  • В первое поле блока select list item перетащите вашу переменную answers из категории Variables.
  • Для выбора случайного индекса элемента нам нужно сгенерировать случайное число. В категории Math найдите блок random integer from 1 to (случайное целое число от 1 до).
  • Перетащите этот блок во второе поле блока select list item.
  • Нам нужно узнать количество элементов в списке ответов. В категории Lists найдите блок length of list (длина списка).
  • Перетащите блок length of list во второе поле блока random integer.
  • В блок length of list добавьте переменную answers.
  • Теперь нам нужно отобразить выбранный ответ. В левой панели найдите компонент Label2 (метка для ответа) и выберите блок set Label2.Text to (установить текст метки).
  • Перетащите блок set Label2.Text to внутрь блока when AccelerometerSensor1.Shaking.
  • Перетащите ранее созданный блок select list item в поле блока set Label2.Text to.
  • Теперь запрограммируем кнопку для тестирования на эмуляторе. Найдите компонент Button1 и выберите блок when Button1.Click (при нажатии на кнопку).
  • Скопируйте содержимое блока when AccelerometerSensor1.Shaking (блок set Label2.Text to с вложенными блоками) и вставьте его в блок when Button1.Click.
Важно: Убедитесь, что в блоке random integer начальное значение установлено как 1, а не 0, поскольку индексы в списках MIT App Inventor начинаются с 1, а не с 0.

Шаг 5: Тестирование приложения

  • Протестируйте ваше приложение:
    Чтобы увидеть ваш магический шар в действии, воспользуйтесь руководством по тестированию приложений в нашем введении в MIT App Inventor.
  • Тестирование на эмуляторе:
    • Запустите приложение на эмуляторе
    • Нажмите кнопку "Встряхнуть (для эмулятора)"
    • Проверьте, что на экране появляется случайный ответ из вашего списка
  • Тестирование на реальном устройстве:
    • Подумайте о вопросе, на который вы хотите получить ответ
    • Встряхните телефон
    • Проверьте, что на экране появляется случайный ответ из вашего списка
    • Попробуйте встряхнуть телефон несколько раз, чтобы увидеть разные ответы
Поздравляем! Вы создали приложение "Магический шар"! Теперь у вас есть цифровой советчик, который будет отвечать на ваши вопросы простым встряхиванием телефона.
Идеи для улучшения:
  • Добавьте звуковой эффект при появлении ответа
  • Создайте анимацию для шара (например, вращение)
  • Добавьте поле для ввода вопроса
  • Создайте разные категории ответов (позитивные, негативные, нейтральные)
  • Добавьте возможность сохранять историю вопросов и ответов
