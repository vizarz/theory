LinearProgress

Компонент отображает горизонтальную полосу прогресса, которая визуализирует выполнение задачи (например, загрузки, обработки данных или отслеживания этапов). Полезен для информирования пользователя о текущем статусе длительных операций.

    Методы

    IncrementProgressBy(value)
    Увеличивает значение прогресса прогресс-бара на указанную величину.

    Свойства

    Indeterminate
    Указывает, находится ли прогресс-бар в режиме неопределенности.
    IndeterminateColor
    Изменяет цвет прогресс-бара в режиме неопределенности.
    Maximum
    Устанавливает верхнюю границу диапазона прогресс-бара (максимальное значение).
    Minimum
    Устанавливает нижнюю границу диапазона прогресс-бара (минимальное значение). Функция работает только для устройств с API >= 26.
    Progress
    Возвращает текущее значение прогресса прогресс-бара.
    ProgressColor
    Изменяет цвет заполненной части прогресс-бара.
    Visible
    Определяет, должен ли LinearProgress отображаться на экране. Значение true – если элемент виден, и false – если скрыт.
    Width
    Устанавливает горизонтальную ширину LinearProgress в пикселях.
    WidthPercent
    Устанавливает горизонтальную ширину LinearProgress как процент от ширины экрана.

    События

    ProgressChanged(progress)
    Событие, которое указывает на изменение уровня прогресса в прогресс-баре. Возвращает текущее значение прогресса. Если свойство “Indeterminate” установлено в true, то возвращается “0”.
