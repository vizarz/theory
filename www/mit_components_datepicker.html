DatePicker

Кнопка, при нажатии на которую, открывается всплывающее окно, позволяющее пользователю выбрать дату по Григорианскому календарю.

Примечание: Дата и время управляются с помощью методов компонента Clock.

    Методы

    LaunchPicker()
    Запускает диалоговое окно DatePicker. Событие AfterDateSet срабатывает после подтверждения пользователем выбранной даты.
    SetDateToDisplay(year, month, day)
    Позволяет задать дату для отображения при открытии DatePicker. Допустимые значения: месяц – от 1 до 12, день – от 1 до 31.
    SetDateToDisplayFromInstant(instant)
    Позволяет задать дату для отображения при открытии DatePicker, основанную на переданном моменте времени (instant).

    Свойства

    BackgroundColor
    Определяет фоновый цвет DatePicker в виде целого числа формата alpha-red-green-blue. Если установлено изображение, изменение цвета не будет видно, пока изображение не удалено.
    Day
    Возвращает число дня месяца, который был выбран с помощью DatePicker.
    Enabled
    Определяет, должен ли DatePicker быть активным и кликабельным.
    FontBold
    Определяет, должен ли текст DatePicker отображаться жирным. Некоторые шрифты не поддерживают жирное начертание.
    FontItalic
    Определяет, должен ли текст DatePicker отображаться курсивом. Некоторые шрифты не поддерживают курсив.
    FontSize
    Определяет размер шрифта текста DatePicker, измеряемый в sp (безразмерных пикселях).
    FontTypeface
    Задает тип шрифта DatePicker: default, serif, sans serif, monospace или custom. Для пользовательского шрифта загрузите файл .ttf в медиа проекта.
    Height
    Определяет вертикальную высоту DatePicker в пикселях.
    HeightPercent
    Определяет вертикальную высоту DatePicker как процент от высоты экрана.
    Image
    Указывает путь к изображению DatePicker. Если заданы и изображение, и фоновый цвет, видимым будет только изображение.
    Instant
    Возвращает момент времени выбранной даты.
    Month
    Возвращает номер месяца, выбранного с помощью DatePicker.
    MonthInText
    Возвращает название месяца, выбранного с помощью DatePicker.
    Shape
    Определяет форму DatePicker. Допустимые значения: 0 (по умолчанию), 1 (округлая), 2 (прямоугольная) и 3 (овал). Форма не отображается, если используется изображение.
    ShowFeedback
    Определяет, должна ли отображаться визуальная обратная связь при нажатии на DatePicker с изображением.
    Text
    Определяет текст, отображаемый на DatePicker.
    TextAlignment
    Определяет выравнивание текста DatePicker. Допустимые значения: 0 (обычное, например, по левому краю для текста слева направо), 1 (центр) или 2 (противоположное, например, по правому краю для текста слева направо).
    TextColor
    Определяет цвет текста DatePicker в виде целого числа alpha-red-green-blue.
    Visible
    Определяет, должен ли DatePicker отображаться на экране. Значение true – если отображается, и false – если скрыт.
    Width
    Определяет горизонтальную ширину DatePicker в пикселях.
    WidthPercent
    Определяет горизонтальную ширину DatePicker как процент от ширины экрана.
    Year
    Возвращает год, выбранный с помощью DatePicker.

    События

    AfterDateSet()
    Событие, которое срабатывает после того, как пользователь выберет дату в диалоговом окне.
    GotFocus()
    Означает, что курсор переместился на DatePicker, и теперь по нему можно кликнуть.
    LostFocus()
    Означает, что курсор переместился с DatePicker, и теперь по нему нельзя кликнуть.
    TouchDown()
    Означает, что DatePicker был нажат.
    TouchUp()
    Означает, что DatePicker был отпущен.
