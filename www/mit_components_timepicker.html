TimePicker

Компонент TimePicker предоставляет пользователю возможность выбора времени (часы и минуты). При нажатии на компонент открывается стандартный диалог выбора времени Android, который позволяет пользователю выбрать конкретное время.

После выбора времени происходит событие AfterTimeSet, которое можно использовать для выполнения действий на основе выбранного времени. Компонент также предоставляет свойства для получения выбранных часов и минут.

    Методы

    LaunchPicker()
    Открывает диалог выбора времени, чтобы пользователь мог выбрать время.

    Свойства

    BackgroundColor
    Определяет цвет фона компонента TimePicker.
    Enabled
    Определяет, включен ли компонент. Если значение установлено как False, компонент будет отключен и пользователь не сможет с ним взаимодействовать.
    FontBold
    Определяет, отображается ли текст полужирным шрифтом.
    FontItalic
    Определяет, отображается ли текст курсивом.
    FontSize
    Определяет размер шрифта текста, отображаемого в компоненте.
    FontTypeface
    Определяет гарнитуру шрифта для текста.
    Height
    Определяет высоту компонента.
    Hour
    Определяет текущий выбранный час (0-23). Можно установить это свойство, чтобы программно задать час.
    Image
    Определяет изображение, которое будет отображаться на компоненте. Если изображение задано, оно будет отображаться вместо текста.
    Instant
    Определяет текущее выбранное время в виде объекта Instant. Можно использовать для работы с датой и временем в приложении.
    Minute
    Определяет текущую выбранную минуту (0-59). Можно установить это свойство, чтобы программно задать минуту.
    TextAlignment
    Определяет выравнивание текста в компоненте. Возможные значения: 0 (по левому краю), 1 (по центру), 2 (по правому краю).
    TextColor
    Определяет цвет текста, отображаемого в компоненте.
    Visible
    Определяет, видим ли компонент.
    Width
    Определяет ширину компонента.

    События

    AfterTimeSet()
    Событие, которое происходит после того, как пользователь установил время в диалоговом окне выбора времени.
    GotFocus()
    Событие, которое происходит, когда компонент получает фокус.
    LostFocus()
    Событие, которое происходит, когда компонент теряет фокус.
    TouchDown()
    Событие, которое происходит, когда пользователь начинает касаться компонента.
    TouchUp()
    Событие, которое происходит, когда пользователь прекращает касаться компонента.
