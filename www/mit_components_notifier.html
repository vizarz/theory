Notifier

Компонент Notifier – это невидимый компонент, который позволяет отображать различные типы предупреждений, уведомлений и диалоговых окон. Компонент имеет три основных типа сообщений: предупреждение (alert), подтверждение (confirmation) и запрос (prompt).

Предупреждение: Диалоговое окно с сообщением и кнопкой ОК.

Подтверждение: Диалоговое окно с сообщением и двумя кнопками: ОК и Отмена.

Запрос: Диалоговое окно с сообщением, текстовым полем для ввода и кнопками ОК и Отмена.

    Методы

    DismissProgressDialog()
    Закрывает диалоговое окно с индикатором прогресса, ранее отображенное с помощью ShowProgressDialog.
    LogError(message)
    Записывает сообщение об ошибке в журнал устройства Android, используя журнал ошибок.
    LogInfo(message)
    Записывает информационное сообщение в журнал устройства Android, используя информационный журнал.
    LogWarning(message)
    Записывает предупреждающее сообщение в журнал устройства Android, используя журнал предупреждений.
    ShowAlert(message)
    Отображает предупреждение с заданным сообщением.
    ShowChooseDialog(message, title, button1Text, button2Text, cancelable)
    Отображает диалоговое окно выбора с двумя кнопками. В ответ на нажатие кнопки происходит событие AfterChoosing.
    ShowMessageDialog(message, title, buttonText)
    Отображает диалоговое окно с сообщением и одной кнопкой. В ответ на нажатие кнопки происходит событие MessageDialogClosed.
    ShowPasswordDialog(message, title, cancelable)
    Отображает диалоговое окно для ввода пароля. В ответ на нажатие кнопок OK или Отмена происходит событие AfterTextInput.
    ShowProgressDialog(message, title)
    Отображает диалоговое окно с индикатором прогресса. Используйте этот метод для случаев, когда длительная операция препятствует работе приложения.
    ShowTextDialog(message, title, cancelable)
    Отображает диалоговое окно для ввода текста. В ответ на нажатие кнопок OK или Отмена происходит событие AfterTextInput.

    Свойства

    BackgroundColor
    Определяет цвет фона для диалоговых окон.
    NotifierLength
    Определяет продолжительность всплывающего уведомления: 1 (короткое) или 2 (длительное).
    TextColor
    Определяет цвет текста для диалоговых окон.

    События

    AfterChoosing(choice)
    Событие, которое происходит после того, как пользователь выбрал одну из кнопок в диалоговом окне, отображенном с помощью ShowChooseDialog.
    AfterTextInput(response, text)
    Событие, которое происходит после того, как пользователь ввел текст в диалоговом окне, отображенном с помощью ShowTextDialog или ShowPasswordDialog.
    ChoosingCanceled()
    Событие, которое происходит, когда пользователь отменяет диалоговое окно выбора, отображенное с помощью ShowChooseDialog.
    MessageDialogClosed()
    Событие, которое происходит, когда пользователь закрывает диалоговое окно сообщения, отображенное с помощью ShowMessageDialog.
